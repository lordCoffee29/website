Sound Modification

For musicians who want to access the sounds of any instrument they desire, anyone who uses this platform can change the sound of their audio to that of another instrument. Using tools such as FFT and DSP, this tool digitizes and modifies input sound waves to create the desired output.

Chord Generation

In songs, chord progressions tend to follow a static I-IV-vi-V pattern. In order to help vary these progressions, this platform generates chord progressions based on an input melody (audio recording). In the future, this software will implement AI and be trained on Romantic and Contemporary Era chord progressions to provide more nuanced options.

Sound Recognition

To identify unpitched sounds, this project takes an input sound and cross references it with a curated sample base to identify the source of the sound. This project was created as a final assignment for a computing course. Future implementations will use larger datasets of unpitched sounds to create more accurate results.
